# **App Name**: AuraCV

## Core Features:

- Dynamic Career Triad: An interactive interface allowing recruiters to toggle between three professional facets: Pedagogy, Early Childhood, and Commerce.
- AI Career Synchronizer: A smart tool that uses reasoning to dynamically suggest which of Alejandra's soft skills best match a pasted job description.
- Experience Vault: A robust backend system using a database to manage chronologically sorted educational paths and professional missions.
- The Quality-to-Asset Matrix: An interactive visualization that flips identified 'flaws' like sensibility and impatience into high-value professional assets.
- Secure Talent Studio: A private authentication-protected dashboard where the user can update the content of her CV and respond to received contact messages.
- Media Portfolio & Research Hub: Integrated cloud storage hosting publication abstracts, university diplomas, and standardized PDF versions of the resume.
- Direct Engagement Hub: A real-time contact form where inquiries from recruiters are validated and stored immediately for administrative follow-up.

## Style Guidelines:

- Primary color: Sunset Terracotta (#E68364), representing warmth, empathy, and professional grounding.
- Background color: Soft Shell (#FAF6F4), a creamy off-white that feels approachable and human rather than clinical.
- Accent color: Deep Muted Rose (#9E4B5E), providing sophisticated contrast for titles and highlighting specific skill categories.
- Headline font: 'Alegreya' (Serif), reflecting an intellectual and formal academic background; Body font: 'Inter' (Sans-serif), ensuring readability for dense CV items.
- Thin-line organic icons used to distinguish the three main sectors (a graduation cap for pedagogy, a toy for animation, and a cart for commerce).
- A modern 'card-deck' structure for the resume section, where entries slide in or fade smoothly during facet switching.
- Staggered reveal animations for list items to create a rhythmic reading flow when expanding work history.
